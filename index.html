Homework №1 (PIC16F877A)

Download this project as a .zip file Download this project as a tar.gz file

Устройство представляет собой электронный термометр, который отправляет данные о температуре по интерфейсу RS-232 раз в секунду. Данный термометр состоит из микроконтроллера PIC16F877, датчика температуры AD22100, и микросхемы MAX232, которая осуществляет преобразование уровней КМОП и ТТЛ в уровни последовательного интерфеса RS-232.
